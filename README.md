A full-stack AI-powered Resume Analyzer and Interview Preparation Platform.

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- Spring Boot
- PostgreSQL
- JWT Authentication

## Features

- Resume Upload
- ATS Score Analysis
- Keyword Suggestions
- Grammar Check
- Interview Questions
- Dashboard Analytics
